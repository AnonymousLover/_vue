### webpack建构

本项目利用[webpack](http://webpack.github.io/)作为基础的建构打包工具

**1、运行项目基本运行**

```
npm install     //安装模块
```

**2、客户端的启动方式**

client端对应的webpack配置文件为 webpack.dev.config.js

```
npm run wp                  //启动webpack client建构

node static_cache_gzip.js   //启动带gzip的静态服务
```
